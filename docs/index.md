# Cognitive Passphrases Generator Documentation

This site contains the current user manual and changelog for the Cognitive Passphrases Generator.

## Available documentation

- [User Manual](USER_MANUAL.md)
- [Changelog](changelog.md)

## Quick start

Run the generator from the repository root:

```bash
python generate_passphrase.py xspace
```

## Scope

The current implementation supports profile-driven passphrase rendering from local JSON catalogs.
Planned features are documented as future work in the user manual.
